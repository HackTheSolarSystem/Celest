Welcome to Celeste! This handy tool collects SDO image files and metadata for use in OpenSpace. After selecting and downloading the desired data, extract the zip file into your Solar Data folder (by default ccmc_files/solarbrowsingdata/event//imagedata///

/). Then navigate to /data/assets/sunearthday/solarhack.scene and change the "local SolarImageryBasePath" variable to be the full path of the "solarbrowsingdata" directory indicated above. Enjoy!
